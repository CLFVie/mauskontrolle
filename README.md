## What is Cursor?

Cursor is a next-generation IDE that uses the power of GPT-4 to make a development experience like no other.
Using written prompts, Cursor can write and edit your code, write documentation or unit tests, find and fix bugs and make your code faster and more efficient. It's creative too, offering suggestions on how to implement features or make changes to improve the quality of your code. With the improved capabilites of GPT-4, Cursor's suggestions stretch far beyond one line or method, with Cursor being able to use an entire file as context for it's code. Gone are the days of asking ChatGPT for code, then spending valuable time embedding generic code into your project. Cursor's code is custom to your project, and integrates flawlessly into your workflow to make one coder code at the speed of 10!

While much of the core functionality is in place, the editor is still in an early Beta stage, so bugs and issues are more than likely to occur. During this stage, the editor is free, but a paid plan will likely come in the future when a v1.0 release is ready.

This repository is purely used for issues, as development is closed source. If you have tried Cursor and have found any bugs, or have any ideas or suggestions, create an Issue and we'll see how we can help!
